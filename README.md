## 📦 Installation

1. **Clone the repository**

```bash
git clone https://github.com/your-username/react-portfolio.git
cd react-portfolio
```

2. **Install dependencies**

```bash
npm install
# or
yarn install
```

3. **Run the project**

```bash
npm run dev
# or
yarn dev
```
